# ESTRELLA juego

Copia local del juego ESTRELLA para trabajar sin depender de Replit.

## Abrir el juego

Abre `index.html` en un navegador, o usa `ABRIR-JUEGO.command` en macOS.

## Estructura

- `index.html`: pantalla principal del juego.
- `public/js/main.js`: arranque del juego.
- `public/js/input.js`: controles.
- `public/js/player.js`: jugador.
- `public/js/world.js`, `biomes.js`, `planets.js`: mundo y planetas.
- `public/js/enemies.js`, `enemyAI.js`, `enemyRender.js`: enemigos.
- `public/js/combat.js`, `spaceCombat.js`: combate.
